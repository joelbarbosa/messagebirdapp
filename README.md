# MessageBird app communication
A challenge with React, Redux, Mongo, Node, MessageBird and its enviroments.

# Project
The project is divided into two projects:

```bash

1 - frontend
2 - backend

```

## Installation BackEnd
```bash

$ cd backend
$ npm install 

```

## Tests BackEnd
```bash

$ cd backend
$ npm test 

```

## Run BackEnd
```bash

$ cd backend
$ npm start

http://localhost:3001/ 

```

## Installation FrontEnd
```bash

$ cd frontend
$ npm install

```

## Tests FrontEnd
```bash

* before test frontend, raise the BackEnd with npm start

$ cd frontend
$ npm test 

```

## Run FrontEnd
```bash

$ cd frontend
$ npm start 

http://localhost:3000/

```
